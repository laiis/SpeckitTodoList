# Quickstart: 多行待辦事項輸入 (Multi-line Todo Input)

## 開發者驗證步驟

### 1. 驗證 UI 結構
- 開啟 `index.html`。
- 確認待辦事項輸入欄位已從 `<input type="text">` 變更為 `<textarea>`。
- 確認該欄位具備 `rows="10"`。

### 2. 驗證快捷鍵
- 在輸入框中輸入文字並按下 `Enter`，確認游標換到下一行。
- 按下 `Ctrl + Enter` (或 `Cmd + Enter`)，確認任務被提交/儲存。

### 3. 驗證列表縮略
- 新增一個包含 15 行文字的任務。
- 在主列表中，確認該任務只顯示前 3 行，且末尾有「...」。
- 確認任務列表佈局未因長任務而崩潰。

### 4. 驗證資料持久化
- 重新整理頁面。
- 進入剛才建立的長任務編輯模式，確認換行內容完整保留。
